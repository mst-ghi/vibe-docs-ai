Environment Setup

To initialize the Nocturne Glass profile, ensure your vibe.config.js has the refraction: true flag enabled. This triggers the hardware-accelerated blur engine. For production environments, we recommend a minimum blur radius of 20px to maintain legible contrast ratios.
VibeDocs AI requires Tailwind CSS v3.4+ to support dynamic opacity values within utility classes. If you are on v2.x, please use our @vibe-coding/legacy-adapter to shim the color tokens and elevation levels.

API & Data Architecture

Free-tier API keys are limited to 5,000 requests per hour. For high-frequency "Vibe Streams," we recommend implementing a WebSocket connection to reduce overhead. Check the Network status for real-time throughput metrics.
Hydration mismatches typically occur when the glass layer's dynamic background-blur is calculated on the server. Wrap your glass components in the wrapper provided in the Vibe UI SDK.
